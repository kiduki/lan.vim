# AGENTS.md

このリポジトリで Git 管理下のファイル修正を行う場合、以下を必須とする。

1. スキル確認（MUST）
- 修正作業を開始する前に、利用可能なスキルが存在するかを必ず確認する。

2. 証跡出力（MUST）
- 作業開始時に、確認結果を明示する。
- 少なくとも次を含めること:
  - `checked_skills: ...`
  - `selected_skill: ...`（該当なしの場合は `none`）

3. 未実施時の扱い（MUST）
- 上記 1 または 2 を満たせない場合、勝手に編集を進めてはならない。
- その時点で「このまま実行してよいか」をユーザーに確認し、明示許可を得てから続行する。
